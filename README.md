# TripWise Vacation Planner

A native Android app (Java, Room/SQLite) for planning vacations and the excursions that make up each trip — with search, a database-driven category picker, a combined Trip Report, and a PIN-protected app lock.

- **Download / project page:** https://syauss1.github.io/WGU_Capstone/
- **Latest release APK:** https://github.com/syauss1/WGU_Capstone/releases/latest
- **Full documentation (features, screens, project structure):** [app/README.md](app/README.md)

## Quick start (developers)

```
git clone https://github.com/syauss1/WGU_Capstone.git
```

Open the project root in Android Studio, let Gradle sync, then run the `app` configuration on an emulator or device (Android 8.0 / API 26+).

To build a signed release APK yourself, see the "Building a Signed Release APK" section in [app/README.md](app/README.md) — you'll need your own `app/keystore.properties` (not included in this repo).
