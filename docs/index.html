What it does

TripWise lets you plan vacations and organize the excursions that make up each trip in one place. Create a vacation with a title, hotel, and start/end dates, attach as many excursions as you like, and keep everything validated, searchable, and summarized in a single report — all stored securely on your device.

Vacations & Excursions

Full create/edit/delete for vacations and their linked excursions, with date pickers that guarantee valid dates.

Search

Live search across vacations by title or hotel, and a combined keyword search across the full Trip Report.

Database-driven categories

Excursion categories come from the database, not hardcoded — add a new one right from the form.

Trip Report

A titled, time-stamped report combining every vacation and excursion into one multi-column table.

PIN lock

App-entry PIN protected with a salted SHA-256 hash — the PIN itself is never stored.

Notifications

Optional alerts for vacation start/end dates and excursion dates.

Installing the APK

  1. Tap Download APK above on your Android device (or transfer the file to it).
  2. Open the downloaded file. If prompted, allow your browser or file manager to install unknown apps for this one install — this is expected for an app installed outside the Play Store.
  3. Tap Install, then open TripWise.
  4. Create a PIN on first launch, and you're ready to plan your first trip.
